The Dopamine Insight

Run trials to see the prediction error shift backward from the reward to the earliest predictive cue‚Äîexactly like dopamine neuron recordings.

How Temporal Difference Learning Works

In the brain, time is represented as a sequence of microstates‚Äîeach moment is a distinct state that can carry its own value. When a cue reliably predicts a future reward, TD learning gradually assigns value to each timestep, propagating backward from the reward.

The TD Learning Rule

At each timestep \(t\), the agent computes a prediction error:

\[\delta(t) = r(t) + \gamma \cdot V(t+1) - V(t)\]
Prediction error at time t = Reward at t + Discounted next value ‚àí Current value

Then update the value at t:

\[V(t) \leftarrow V(t) + \alpha \cdot \delta(t)\]

The Backward Shift

The magic happens over trials:

  • Trial 1: Only the reward time has a positive error (unexpected reward)
  • Trial 5: Error starts appearing at Cue 2 (which now predicts reward)
  • Trial 20: Error shifts to Cue 1 (the earliest predictor)
  • Converged: Error only at Cue 1; reward time has Œ¥ ‚âà 0 (fully expected)

This backward shift is exactly what Schultz observed in dopamine neurons: early in learning, dopamine fires at reward; later, it fires at the earliest predictive cue.
